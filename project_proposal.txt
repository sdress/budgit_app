Personal Project Proposal

Description:
A monthly budget app where users can input their expenses into a table, and see how much they have remaining on a calendar. Possibly set a savings goal.

Stack: Python / Django

Feature List:
- login/registration page
- form for adding/editing expenses
- dashboard with expenses table and charts
- Google API calendar & charts

(Time permitting) Extra features:
- form for adding/editing goals
- savings goal progress bar
